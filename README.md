# GraphQL Demo

## Requirements

- Docker
- Docker Compose

## Development

- Initial setup

  - Add the following to your hosts file:

    ```
    127.0.0.1 graphql-demo.test
    ```

  - Run the setup script:

    ```
    ./setup.dev
    ```

- Spin up all necessary Docker containers for development

  ```
  ./start.dev
  ```
